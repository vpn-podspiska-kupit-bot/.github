# .github
UniVPN — это Telegram-бот для быстрой и удобной покупки VPN-подписки. Сервис поддерживает промокод START, который даёт 3 дня тестового использования, чтобы оценить стабильность и скорость подключения. Подходит для тех, кому нужен защищённый трафик, приватность и простой способ управления доступом прямо в Telegram.
